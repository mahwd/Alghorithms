# Installation

Use the node package manager [npm](https://www.npmjs.com/) to install dependencies.

```bash
npm install
```

# Usage

## Fibonacci task
```bash
npm run fibonacci
```
## Elevation map task
```bash
npm run elevation
```
## Ip restore task
```bash
npm run ip
```
## Max profit calculation task
```bash
npm run profit
```
